Frequently Asked Questions and Answers

Stuck loading

Use Firefox, Chrome or Edge - IE is not supported. If it's still stuck loading clear all history cache etc then retry.

How to copy and paste

Within the CLI tab press ("Shift" + "Ctrl" + "Alt") paste text into box, close the text box then right click on the CLI. Copy and paste is now natively supported on the Pro Lab

Export config to PC

1. Right click on node 2. Click export 3. Open the saved config tab from the left hand navigation bar

Sign in prompt

Do you get asked for a usernmane and password when trying to connect to a node? If you do, sign out using the logout button on the topology page and log back in

My node is asking for a password

Nexus - admin (no password), ACS-CLI - admin/Root123, ACS-WEB - QFX root/juniper, acsadmin/Root123 , Juniper vMX (v14) - lab/lab123, Juniper vMX v15, VRR root (no password), ISE admin/Test123 - change console to telnet, Cisco SD-WAN - admin/admin

Juniper Nodes?

The NEW Juniper vMX images are based on dual nodes setup, where Routing engine (VCP) is connected to Forwarding plane (VFP) and act like single node. All the cli configurations will be done on the VCP, but porting and connections will be on VFP. You can use vMX (v14) if you require the older image, this is deployed as a single node
